PointSequence

Represents a sequence of points

A sequence is a group of points that are kept in an ordered list. The order in which the points go in dictates how they are returned in a First In, First Out manner.

This is an immutable implementation of {see PointSequenceInterface}. Linking this sequence with points or other sequence of points returns always a new instance.
